## Project Overview
This project is simply a machine learning system that is meant to predict a student’s performance based on their lifestyle habits, which include study time, sleep, attendance, screen time, and exercise.
The model uses the following categories for a student’s performance:
- Poor
- Average
- Good
If the performance is not up to standard, recommendations on how to improve are provided.

## Features
- Predicts student performance using ML
- Takes real-time user input
- Provides personalized improvement suggestions
- Simple and beginner-friendly implementation

## How It Works?
1. A dataset is created with student lifestyle data
2. Data is split into training and testing sets
3. A Decision Tree model is trained
4. User inputs their own data
5. Model predicts performance
6. Improvement tips are shown if needed
